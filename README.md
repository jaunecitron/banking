## Requirement

- [node](https://nodejs.org/en/) >= `v12.18.2`

## Install

```bash
$ git clone git@github.com:jaunecitron/banking.git
$ cd banking
$ npm install
```

## Run locally

```bash
$ npm run start:local
# Watch mode
$ npm run start:watch
```

## Test

```bash
$ npm test
# TDD
$ npm run test:watch
```
